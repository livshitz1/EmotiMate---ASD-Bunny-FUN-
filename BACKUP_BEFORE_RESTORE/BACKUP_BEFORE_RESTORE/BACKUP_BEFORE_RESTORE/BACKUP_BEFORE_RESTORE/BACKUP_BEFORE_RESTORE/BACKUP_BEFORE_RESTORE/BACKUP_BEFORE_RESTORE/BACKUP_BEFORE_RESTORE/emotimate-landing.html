מה זה EmotiMate?

EmotiMate היא אפליקציה טיפולית רגשית עם ארנב מתוק שמגיב לרגשות הילד, מציע משימות יומיות פשוטות, ויוצר תחושת קשר וביטחון – בבית, בגן, או בטיפול.

איך זה עובד?

  • 🐰 הארנב מגיב ומדבר בעדינות
  • ✅ מוצגת משימה מתאימה
  • 📊 הורה או מטפל מקבל תיעוד פשוט
